# 🎬 Movie Recommender AI (Pro Version)

A sophisticated ❯ Machine Learning ❮ project that recommends movies based on user preferences. This end-to-end application uses Natural Language Processing (NLP) to suggest movies and fetches real-time data like posters and trailers from the ❯ TMDB API ❮.

## 🚀 Features
- [ Content-Based Recommendation ] : Suggests movies similar to the ones you like.
- [ Real-time Data ] : Fetches movie posters and ratings dynamically.
- [ Integrated Trailers ] : Watch official YouTube trailers directly within the app.
- [ Search by Genre ] : Explore movies across different categories.
- [ Interactive UI ] : A clean, dark-themed interface built with Streamlit.

## 🛠️ Tech Stack
- [ Language ] : Python 3.10+
- [ Machine Learning ] : Scikit-learn (CountVectorizer, Cosine Similarity)
- [ Data Handling ] : Pandas, Numpy, Pickle
- [ Frontend ] : Streamlit
- [ API ] : The Movie Database (TMDB)

## 📁 Project Structure
- ➔ app.py: The main Streamlit web application.
- ➔ movie_recommender.ipynb: Jupyter Notebook containing data preprocessing and model training.
- ➔ movie_dict.pkl: Serialized movie data.
- ➔ similarity.pkl: Pre-computed similarity matrix.

## ⚙️ How to Run Locally
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
